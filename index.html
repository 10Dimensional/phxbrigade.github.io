Code for Phoenix

What is Code for America?

Code for America is a group of people, first and foremost. We also call ourselves technologists, designers, planners, doers, and activists.

What is the Code for Phoenix Brigade?

Code for Phoenix is a local chapter of the Code for America Brigade, an international network of people committed to using their minds, voices and hands, in collaboration with local governments, to make their cities better.

The Vision

Code for Phoenix is a safe space for experimentation, learning and practicing skills, taking chances, and asking questions, all to affect and encourage positive change. Whenever possible, we proactively reach out to those who are left out, excluded, ignored, or discriminated against. Those people, in our opinion, represent our most desirable source for experiences, knowledge, and thoughts. We fail without their involvement.
